## 一、项目介绍

语言: Java  数据库:MySQL


后端技术：SpringBoot、Mybatisplus

前端技术：Vue、ElementUI

本系统包含用户和管理员两个角色

用户：登录、注册、物品发布、消息模块、物品购买模块、收藏模块、下架模块、售出模块、购买模块、个人中心等。

管理员：登录、用户信息管理、商品信息管理、订单信息管理等
